# Landing Page - GreenTech

## Objetivo
Desenvolver uma landing page para a GreenTech, uma startup de tecnologia sustentável, para atrair visitantes e capturar leads.

## Escopo
- Header fixo com navegação
- Seção inicial com chamada para ação (CTA)
- Formulário de contato funcional
- Design responsivo básico

## Tecnologias
- HTML5, CSS3, JavaScript (vanilla)

## Gestão do Projeto

### Metodologia
- **Framework:** Kanban via GitHub Projects
- **Sprints:** Milestones de 1 semana cada
- **Planejamento:** Projects com colunas Backlog, Em Progresso, Concluído
- **Visualização:** Filtro por Milestone para foco na sprint ativa

## Como Visualizar
